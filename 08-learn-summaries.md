# Microsoft Learn summaries (DP-203)

One-line distillations of each Learn module.

- **Storage design**: Lake vs warehouse, files (Parquet/Delta/Avro), partitioning, serving layer.
- **Partition strategy**: Pruning, efficiency, parallelism.
- **Serving layer**: Star schema, SCDs, hierarchy, temporal.
- **Physical storage**: Compression, partition, sharding, distribution, redundancy.
- **Logical structures**: Temporal, SCDs, external tables, hierarchies.
- **Ingest + transform**: Batch + stream, incremental loads, late data.
- **Batch**: Synapse / ADF / Mapping Data Flows / Spark, retention, upsert.
- **Stream**: Stream Analytics, Structured Streaming, windowing, event vs processing time.
- **Pipelines**: Schedule, version control, notebooks, errors/retries.
- **Security**: Masking, encryption (TDE/CMK), RBAC, ACLs, RLS/CLS, Purview, identities.
- **Monitor**: Azure Monitor logs/metrics, pipelines, queries, clusters.
- **Optimize**: Small files, formats, partitioning, distribution, skew, statistics.
